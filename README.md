# REDCORE SYSTEMS | RCS-01 Sentinel

This project is a futuristic website concept created for the COS003 Introduction to Social and Digital Media module. The website presents **RCS-01 Sentinel**, a fictional autonomous robotic security unit designed for surveillance, threat detection, and high-risk environment control.

The main purpose of this website is to demonstrate basic web design skills, digital media integration, content organisation, and a clear visual concept. The website uses a dark sci-fi style with red visual accents to create a technological and immersive atmosphere.

## Website Sections

- Home
- About
- Technology
- Specs
- Logs
- Request Access

## Skills Demonstrated

- HTML structure
- CSS styling
- Website layout design
- Navigation menu creation
- Digital media integration
- Visual branding
- Content organisation

## Project Purpose

This project reflects my interest in IT, web design, digital technologies, and future software development. It helped me practise creating a structured website and presenting a digital concept in a professional and creative way.
